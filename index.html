Contenedor y maya en Bootstrap

Contenedores: Son un bloque de construcción fundamental de Bootstrap que contienen, rellenan y alinean su contenido dentro de un dispositivo o ventana gráfica determinada.

Cómo funcionan: Los contenedores son el elemento de diseño más básico de Bootstrap y son necesarios cuando se utiliza nuestro sistema de rejilla por defecto. Los contenedores se utilizan para contener, rellenar y (a veces) centrar el contenido que contienen. Aunque los contenedores pueden estar anidados, la mayoría de los diseños no requieren un contenedor anidado. Bootstrap viene con tres contenedores diferentes:
.container, que establece un ancho máximo en cada punto de respuesta,
.container-fluid, que es width: 100% en todos los puntos de ruptura,
.container-{breakpoint}, que es width: 100% hasta el punto de ruptura especificado.

Sistema de maya: Se utiliza la maya para construir diseños de todas las formas y tamaños gracias a un sistema de doce columnas. El sistema de maya de Bootstrap utiliza una serie de contenedores, filas y columnas para diseñar y alinear el contenido. Está construido con flexbox y es totalmente responsivo.
